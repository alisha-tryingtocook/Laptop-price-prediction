Laptop Price Prediction using Linear Regression

This project builds a machine learning model to estimate laptop prices using technical specifications such as RAM, CPU speed, storage capacity, screen resolution, and device weight. The dataset required extensive preprocessing, including feature extraction from messy text fields, missing value handling, and one-hot encoding for categorical variables.

Exploratory data analysis and visualisation were performed to understand pricing trends and feature correlations. A linear regression model was trained and evaluated using RMSE, MAE, and R², achieving strong predictive performance on unseen data.

Tech Stack:
Python • pandas • NumPy • scikit-learn • matplotlib • seaborn • Jupyter Notebook
